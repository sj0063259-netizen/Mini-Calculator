🧮 Mini Calculator (JavaScript Project)

A simple and responsive calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations and demonstrates the use of DOM manipulation and event handling.

---

🚀 Features

- Basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (×)
  - Division (/)
- Clear button to reset the display
- Responsive design (works on laptop and mobile screens)
- Interactive button clicks using JavaScript
- Clean UI with CSS Grid layout

---

🛠 Technologies Used

- HTML5 – Structure of the calculator
- CSS3 – Styling and responsive layout
- JavaScript (DOM) – Handling button clicks and calculations

💻 How It Works

1. Buttons are selected using JavaScript ("querySelectorAll").
2. Each button click updates the calculator display.
3. When the "=" button is pressed, JavaScript evaluates the expression.
4. The "C" button clears the display.
